Digital Eyewear

These samples demonstrate how to build apps for the different classes of digital eyewear devices.

Stereo Rendering sample shows how to render a stereo view on the following optical see-through devices: ODG R-6, ODG R-7 and Epson BT-200.

AR/VR sample shows how to create a mixed reality app for a video see-through device.

Best Practices

These samples show how to build good user experience design principles into apps, especially when a feature requires the user to perform a specific behavior in order to use the feature.

Books sample is a book store app that enables users to scan book covers for more information on the title. It shows how to control the lifecycle of a Cloud Recognition app and how to direct users to use Cloud Recognition apps effectively.

Penguin sample shows how to guide user behavior to initialize Smart Terrain and how to scan the stage of a Smart Terrain experience.
